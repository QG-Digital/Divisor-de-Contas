O que é este sistema?

O Divisor de Contas é uma aplicação web completa desenvolvida para simplificar e organizar o gerenciamento de despesas compartilhadas entre grupos de pessoas. É a solução perfeita para quem divide moradia, faz viagens em grupo, ou precisa administrar gastos coletivos de forma justa e transparente.

Moradores

Aluguel, contas de luz, água, internet, mercado

Grupos de Viagem

Hospedagem, alimentação, transporte, passeios

Famílias

Contas da casa, compras do mês, despesas comuns

Colegas de Trabalho

Almoços, coffee breaks, eventos

Como Funciona na Prática

Exemplo de Uso

Situação: 3 amigos morando juntos
Cadastro: Ana, Bruno e Carlos são registrados no sistema
Despesas do mês:
  • Ana paga aluguel: R$ 1.500
  • Bruno paga mercado: R$ 600
  • Carlos paga internet: R$ 120
Resultado automático:
  • Sistema calcula que cada um deve R$ 740 no total
  • Ana já pagou R$ 1.500 → deve receber R$ 760
  • Bruno pagou R$ 600 → deve pagar R$ 140
  • Carlos pagou R$ 120 → deve pagar R$ 620
Solução: Bruno paga R$ 140 para Ana, Carlos paga R$ 620 para Ana
